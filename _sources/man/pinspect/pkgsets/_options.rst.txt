Positional Arguments
--------------------

:pkgset:  
        pkgset to inspect

Options
-------

--all  
       display info on all pkgsets

Base Options
------------

-h, --help       
                 show this help message and exit

--debug          
                 enable debugging checks

-q, --quiet      
                 suppress non-error messages

-v, --verbose    
                 show verbose output

--color BOOLEAN  
                 enable/disable color support
