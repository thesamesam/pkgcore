Base Options
------------

-h, --help       
                 show this help message and exit

--debug          
                 enable debugging checks

-q, --quiet      
                 suppress non-error messages

-v, --verbose    
                 show verbose output

--color BOOLEAN  
                 enable/disable color support

Subcommand Options
------------------

-r REPO, --repo REPO  
                      target repo

Subcommands
-----------

profile commands
