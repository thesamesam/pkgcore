Config Options
--------------

--domain DOMAIN  
                 custom pkgcore domain to use for this operation

Base Options
------------

-h, --help       
                 show this help message and exit

--debug          
                 enable debugging checks

-q, --quiet      
                 suppress non-error messages

-v, --verbose    
                 show verbose output

--color BOOLEAN  
                 enable/disable color support

Subcommand Options
------------------

--skip-ldconfig  
                 do not update etc/ldso.conf and ld.so.cache
